## How to use it
### 1.install
```
npm install -g anydoor
```

### 2.instructions

```
anydoor #set current directory as root of static server

anydoor -p 8080 #set port number for server

anydoor -h localhost # set host as localhost

anydoor -d /usr # set root directory as /usr

```
